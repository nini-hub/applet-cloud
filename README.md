### 云开发 quickstart
本项目为使用云开发，结合百度智能云图像识别，开发的图像识别小程序（仅用于学习）

 - [x] 首页
   - [x] 轮播图
   - [x] 企业介绍
   - [x] 导航栏
 - [x] 美食新闻
   - [x] 新闻列表
   - [x] 新闻详情
 - [x] 菜品识别
   - [ ] 菜品拍摄识别，图像识别
   - [ ] 识别结果（菜品名称，置信度，卡路里含量，百科信息）
